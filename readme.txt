Trabalho Prático 4 - Linux e GIT
Objetivo: Este exercício tem como objetivo a interação com o Sistema Operacional e a prática de comandos no terminal do Linux, e também controle de versão Git e prática com o Github.

I Instruções
Utilizar o Webminal para a prática dos comandos Linux;
Realizar os exercícios dispostos nos slides de Linux;
Salvar os resultados em um arquivo.txt, imagem (tirando print), ou outro, como preferir; 
Criar um repositório público no Github para a entrega do Trabalho Prático 4, realizando o commit dos resultados gerados anteriormente;
Enviar o link do repositório como entrega da tarefa.
